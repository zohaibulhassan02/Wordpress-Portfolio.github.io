== Description ==

= WordPress Click to Call Now Button Plugin =

We believe that you shouldn't have to hire a developer to create a click to call button in WordPress. That's why we built WP Call Button, a click to call button plugin for WordPress that's EASY and POWERFUL.

WP Call Button allows you to add a beautiful call now (click-to-call) button on your website with just a few clicks, no coding skills needed.

User experience is our #1 priority with WP Call Button. That's why we have added plenty of customization options to make it easy for beginners and non-techy users to personalize their click to call buttons to match their website brand and color schemes.

WP Call button is a 100% mobile responsive which means that it will always look great on all devices (mobile, tablet, laptop, and desktop). You also have the option to only show the call button on mobile devices only!

WP Call button plugin is also highly optimized for web and server performance because we understand the importance of speed when it comes to SEO, marketing, and conversion. We can honestly say that WP Call Button plugin is one of the most lightweight and fastest WordPress call button plugins in the world.

= Beginner Friendly Call Button - Easy for Non-techies =

We took the pain out of creating click to call buttons and made it easy. Simply enable the plugin, enter your phone number, and that's it!

To make sure there are no typos and the call now button works globally, we have even added an easy country picker, so you always have the right country code in the click to call link.

You have the option to customize the call button text, choose where you want to display it (inside page content, sidebar widgets, or sticky floating button everywhere on the site), and you can even show the call button only to mobile users.

= Click to Call Button that's Optimized for Conversions =

As business owners, we know it's extremely important to book as many phone calls and close as many leads possible to grow the business.

That's why we made WP Call button optimized for high conversions.

By default, you have the option to add a smart floating call button that scrolls with the user as they browse your website. You can choose from our four high-converting sticky call button layouts.

These layout makes sure that your click to call button is highly visible to users which leads to more phone calls for your sales team!

Best part is that we have built-in event tracking that works seamlessly with Google Analytics, so you know which pages are generating you the most calls, so you can focus on what's working to grow your business even faster!

= Call Now Button Generator, Integrations, and More =

Aside from the smart flotaing call button, we also have a click to call button generator that lets you add quick call buttons anywhere on your website including individual posts, pages, and sidebar widgets.

With our call now button widget, you can easily add a call to request a quote area on your sidebar with a prominent click to call button, so you can make the phone ring!

Our static call button generator shortcode works seamlessly with the <a href="https://www.wpbeginner.com/plugins/how-to-disable-gutenberg-and-keep-the-classic-editor-in-wordpress/">Classic Editor</a>. If you're using <a href="https://www.wpbeginner.com/beginners-guide/how-to-use-the-new-wordpress-block-editor/">Gutenberg</a>, then you can use our built-in WP Call Button Block to easily create and customize beautiful click to call buttons within minutes.

WP Call button seamlessly integrates with all <a href="https://www.wpbeginner.com/showcase/best-business-phone-services/">top business phone services</a> across the world.

We also made sure that WP Call button works seamlessly with other popular WordPress plugins such as WPForms, Elementor, WooCommerce etc.

Whether you want to add a call button to Elementor, click to call button with WooCommerce, or a simple call button on your WPForms contact page, this plugin makes it easy!

== Installation ==

1. Install WP Call Button either via the WordPress.org plugin repository or by uploading the files to your server. (See instructions on <a href="http://www.wpbeginner.com/beginners-guide/step-by-step-guide-to-install-a-wordpress-plugin-for-beginners/" rel="friend">how to install a WordPress plugin</a>)
2. Activate WP Call Button.
3. Navigate to the WP Call Button settings page from the Settings Menu of WordPress to enable the click to call button.

== Frequently Asked Questions ==

= Why should I use WP Call Button? =

WP Call Button helps you add quick click now button on your website, so your website visitors can easily call your business. Our smart click to call links help users avoid calling the wrong number and ensures that your business get maximum leads.

= Do I need to have coding skills to use WP Call Button? =

Absolutely not. You can create and manage quick call buttons without any coding knowledge. WP Call Button is the most beginner friendly click to call button plugin in the market.

= Do I need to start the phone number with country code? =

WP Call button comes with a smart phone field, so you don't need to enter the country code. Simply select your country from the dropdown, and we will format the code accordingly.

= Can I change the appearance of the Call Now Button? =

Yes. You can easily customize the color, text, placement, and everything else related to the call now button.

= Can I show the call now button only to mobile users? =

Yes. We have the option to only display call now button to mobile users only.
